## 변수 
* 프로그래밍의 첫번째 작업은 필요한 데이터들을 반드시 **변수(저장소)**로 저장해야 한다.
## 변수 작성법
* `변수선언키워드 생성변수명;` -> 빈 저장소 생성
    * `var box`
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box = 1;`
* `생성변수명 대입연산자 대입값;`
    * `box = 2;`
* 변수선언키워드 : var, let, const
* 생성변수명 - 영문, 숫자조합(숫자는 영문뒤로)
## 객체, 속성, 메서드 작성법
* `객체;` -> 객체 읽음
* `객체.속성;` -> 객체의 속성 읽음
* `객체.메서드();` -> 객체를 메서드방법으로 실행함
* `객체.속성.메서드();`
* `객체.속성.속성.메서드();`
* `객체.메서드().속성;`
* `객체.메서드().메서드();`
## 데이터타입 구분 함수 `typeof()`
* css의 배경색상으로 영역구분하는것처럼 데이터타입 구분목적 사용
1. 특정 데이터타입을 담아둔 변수를 생성
    * `let num = 1;`
    * `let str = '1';`
2. 해당 변수의 데이터타입을 구분하기 위해 typeof 함수 확인
    * `console.log(typeof(num));` //Number(숫자)
    * `console.log(typeof(str));` //String(문자)
## 변수 작성 시 주의사항
* 변수의 이름은 문자, 숫자, 달러, 밑줄이 포함.
* 대시(-), 마침표(.) 사용불가
* 키워드, 예약어 사용 불가 (delate, this, false, true, if, default, var, new, class 등)
* 모든 변수는 대/소문자 구별. Score 과 ,score 은 다른변수
* 변수가 저장할 정보의 종류를 표현하는 의미 있는 단어를 사용해야함. 
## 변수 표기법 종류
* 캐멀 표기법 : `newName, webSiteCode`
* 파스칼 표기법 : `NewName, WebSiteCode`
* 밑줄 표기법 : `new_name, web_site_code`
## 
* 클래스 내부에 선언된 함수는 메소드라 부르며, 클래스와 관련없다면 일반적으로 전부 함수라고 부른다. -> `실행`
* `async` 
    페이지 로드 관계없이 다운로드 후 즉시 실행 스크립트
* `defer` 
    페이지 로드가 완료된 후 즉시 실행 스크립트
    외부 스크립트 시에만 사용 가능 
    주로 위에 많이 씀 (defer 없이 html 동적 js를 아래 파일에서 작성한다면 에러 남)
* `대입연산자(=)`
    변수에 값을 저장할 때 사용하는 연산자
    할당연산자라고도 불린다.
